# VIP Twin
## Overview
A privacy-first AI digital twin that builds long-term memory from a user’s personal data to provide personalized insights and decision support.

## Problem
People lose context across notes, documents, and conversations. Existing AI tools are stateless and require users to constantly re-explain themselves.

## Solution
This project explores a user-owned AI system that retains context across sessions and improves through continued use.

## Current Status
Early MVP / active development.
Core functionality includes local LLM inference, document ingestion, and vector-based memory retrieval.

## Roadmap (Next 12 Weeks)
- User-facing MVP
- Early user onboarding
- Engagement and retention validation

## Tech Stack
- Python / FastAPI
- Local LLMs (Ollama)
- Vector embeddings & retrieval
- Local-first architecture
